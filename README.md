# QuizApp

Full-stack web application for creating and passing quizzes.

Проект представляет собой платформу для создания, управления и прохождения интерактивных квизов.

---

# Возможности

## Пользовательская часть

- Регистрация пользователей
- Авторизация через JWT
- Личный кабинет
- Просмотр доступных квизов
- Прохождение квизов
- Просмотр информации профиля


## Организаторская часть

- Создание квизов
- Редактирование квизов
- Удаление квизов
- Добавление вопросов
- Добавление вариантов ответов
- Управление содержимым квиза


---

# Архитектура проекта

Проект разделён на две части:


## Backend

Технологии:

- Python
- Django
- Django REST Framework
- JWT Authentication
- SQLite


Backend отвечает за:

- пользователей
- авторизацию
- базу данных
- API
- работу с квизами


Структура:
backend/

config/
настройки Django проекта

users/
регистрация
авторизация
пользователи

quizzes/
квизы
вопросы
ответы

rooms/
комнаты прохождения



---

## Frontend

Технологии:

- React
- Vite
- Axios
- React Router


Frontend отвечает за:

- интерфейс
- страницы
- взаимодействие пользователя с API


Структура:
frontend/

src/

components/
переиспользуемые компоненты

pages/
страницы приложения

services/
запросы к API

router.jsx
маршрутизация

# Запуск проекта


## Backend


Перейти:


cd backend



Установить зависимости:


pip install -r requirements.txt



Запустить:


python manage.py runserver



Backend будет доступен:


http://127.0.0.1:8000




---

# Frontend


Перейти:


cd frontend



Установить зависимости:


npm install



Запустить:


npm run dev



Frontend будет доступен:


http://localhost:5173



---
